Инструкция к скрипту stalker.py.

Скрипт для удаление сохранений на кириллице в игре S.T.A.L.K.E.R.
Удаляет ненужные сохранений, которые сохранились на кириллице.

Как пользоваться?
Просто запускаем stalker.py файл, и всё.
Скрипт сам удаляет все файлы на кириллице.

Файлы, то есть, сохранений(сейвы) удаляются по директорию:
D:\Games\games_ready\S.T.A.L.K.E.R. - Объединенный Пак 2\appdata\savedgames
